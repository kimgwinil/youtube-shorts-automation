# 배경 프롬프트 묶음

아래 프롬프트는 이미지 생성 도구나 영상 생성 도구에 바로 넣을 수 있도록 작성했습니다.

사용 원칙:

- 출력 비율: `9:16 세로형`
- 중앙 하단에 자막이 들어갈 여백 확보
- 인물은 정면 클로즈업보다 `작게`, `측면`, `뒷모습` 위주
- 글귀보다 배경이 과하게 튀지 않게 `고요하고 절제된 화면`
